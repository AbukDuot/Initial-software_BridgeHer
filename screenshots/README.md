## How to Add Screenshots

1. Take screenshots of the features listed above
2. Save them with the exact filenames shown
3. Place them in this `screenshots/` directory
4. The README.md will automatically display them

## Screenshot Guidelines

- Use high-quality PNG format
- Capture full feature context (not just isolated elements)
- Include relevant UI elements (navbar, buttons, etc.)
- For email screenshots, use actual email client (Gmail, Outlook, etc.)
- For responsive views, use browser DevTools or actual devices
